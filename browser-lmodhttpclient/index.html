HTTP Request from inside WASM implemented in ofunc/lua

This example uses erdian718/lmodhttpclient (aka ofunc/lmodhttpclient) library for erdian718/lua (aka ofunc/lua).

See the output in browser developer console.

Actual code:


local io = require 'io'
local httpclient = require 'http/client'

local res = httpclient.get('https://httpbin.org/anything')
io.copy(res, io.stdout)
res:close()
